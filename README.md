# toxicoder.github.io

This repository hosts the personal GitHub Pages site for [Alex Abfalter (toxicoder)](https://toxicoder.github.io).

The site is built using [Jekyll](https://jekyllrb.com/) and serves as a portfolio and contact page.

## Local Development

To run this site locally:

1.  Ensure you have Ruby and Bundler installed.
2.  Install dependencies:
    ```bash
    bundle install
    ```
3.  Serve the site:
    ```bash
    bundle exec jekyll serve
    ```
